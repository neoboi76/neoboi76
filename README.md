A desktop-based music player built with Java that integrates GUI, database persistence, and audio playback following the MVC architecture pattern for modularity and maintainability.

**Key Features:**
- Music playback with play, pause, and stop controls
- Song library management with table-based interface for adding, displaying, and removing songs
- Database integration with JPA for persistent song storage
- Custom event handling system (SongEvent, SongListener) for UI responsiveness
- Multithreaded audio playback via PlayerThread for smooth performance
- Clean Swing-based GUI with modular panels (ControlPanel, SongPanel, TablePanel)

**My Contributions (Lead Backend Developer):**
- Implemented MySQL-backed JPA (Hibernate) persistence layer for song metadata and user state management
- Designed and developed the Model layer with Database.java and SongEntity.java
- Created multithreaded playback engine using custom event/listener patterns for responsive UI
- Developed full CRUD operations with robust error handling
- Ensured seamless integration between UI components and backend services
- Configured JPA persistence settings via persistence.xml

**Tech Stack:** Java SE, Java Swing, JPA/Hibernate, MySQL, Multithreading, MVC Architecture
